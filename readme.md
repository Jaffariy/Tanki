Tank Battle
Танковый бой - это игра, разработанная с использованием библиотеки Pygame. В этой игре игроки могут сражаться друг с другом в танках на виртуальном поле битвы. 
Цель игры - уничтожить танк противника, контролируя свой собственный танк и стреляя по целям.

Особенности:
Реалистичная физика движения и поведения танков
Разнообразные уровни, каждый с уникальным ландшафтом и сложностью
Три режима игры: для одного игрока, для двух игроков, режим "Выживание"
Возможность модифицировать и улучшать свой танк с помощью заработанных баллов или денег
Звуковые эффекты и музыкальное сопровождение, создающие атмосферу сражения

Установка
Клонируйте репозиторий на свой компьютер:

git clone https://github.com/Jaffariy/Tanki
Убедитесь, что у вас установлен Python 3.7 и выше.

Установите зависимости с помощью команды:

pip install -r requirements.txt
Запустите игру с помощью команды:
python main.py
